# ShatRanj Frontend ♛

The frontend application for ShatRanj, built using React, Vite, and TypeScript.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, start the local development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser to see the result.

## Production Build

To compile and bundle the application for production:

```bash
npm run build
```

The output files will be built into the `dist/` directory, which can be served using any static host.
